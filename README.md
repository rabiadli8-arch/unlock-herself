# Unlock Herself landing page

Luxury/elegant conversion landing page for a lifestyle, health and wellbeing coach.

## Notes

Instagram and the supplied Google Drive folder were not accessible from the agent runtime without login/access. This first version therefore uses premium placeholder photography and inferred positioning from the handle/name and user brief.

## Conversion strategy

- Soft luxury hero positioning
- Low-friction primary CTA: free clarity call
- Trust-building method: Reset / Restore / Refine / Rise
- Premium offer architecture: Clarity Call + Unlock Audit
- Netlify lead form: `unlock-herself-intake`
